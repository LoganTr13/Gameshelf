# 🎮 Game Bookshelf

Uma estante virtual de jogos feita para uso pessoal e em lives.

***
O site organiza os jogos em categorias como:
- Zerados
- Em live
- Off-live
- Backlog
- Abandonados

Os jogos aparecem inicialmente apenas pela lateral da capa, simulando uma estante física de jogos/DVDs.  
Ao passar o mouse sobre um jogo, sua capa frontal é exibida.

O projeto foi feito utilizando HTML, CSS e JS puro, utilizando JSON para armazenamento dos dados e uma estrutura modular baseada em POO visando facilitar manutenção e expansão futura.
